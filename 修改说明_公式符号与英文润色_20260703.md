## 6. 双模推进间接法推导

位置：主 `.tex` 文件约第 137 到 223 行。

主要核对和修改如下。

### 6.1 性能指标和 Hamiltonian

将性能指标说明改写为更严谨的形式，强调燃料最优目标保留两种模式各自的 `T_i/c_i` 项，而不是简单的节流积分。

Hamiltonian 保持如下形式：

```latex
H=\bm{\lambda}_x^{\mathrm{T}}\left[\frac{T_1\delta_1+T_2\delta_2}{m}\bm{M}(\bm{x})\bm{\alpha}+\bm{D}(\bm{x})\right]
 +(1-\lambda_m)\left(\frac{T_1}{c_1}\delta_1+\frac{T_2}{c_2}\delta_2\right)
```

该形式与质量方程 `\dot m=-T_1\delta_1/c_1-T_2\delta_2/c_2` 一致。

### 6.2 最优推力方向

核对后保留最优方向：

```latex
\bm{\alpha}^{*}=-\frac{\bm{M}^{\mathrm{T}}(\bm{x})\bm{\lambda}_x}{\left\|\bm{M}^{\mathrm{T}}(\bm{x})\bm{\lambda}_x\right\|}
```

该式与 primer vector 反方向规律一致。

### 6.3 开关函数符号方向

重点修正了代入最优方向后的 Hamiltonian 符号关系：

```latex
H=\bm{\lambda}_x^{\mathrm{T}}\bm{D}(\bm{x})-\delta_1S_1-\delta_2S_2
```

并明确说明：在当前符号约定下，`S_i>0` 表示激活第 `i` 个模式会降低 Hamiltonian，因此该模式在最小值原理下是有利的。

对应开关函数为：

```latex
S_i=\frac{T_i}{c_i}\left(\frac{c_i}{m}\left\|\bm{M}^{\mathrm{T}}\bm{\lambda}_x\right\|+\lambda_m-1\right),\qquad i=1,2
```

修改原因：原文中如果写成 `+\delta_i S_i`，则会与后续 `S_i>0` 对应开机的判据冲突。

### 6.4 双曲正切平滑

保留平滑形式：

```latex
\delta_i=\frac{1}{2}\left[1+\tanh\left(\frac{S_i}{\varepsilon}\right)\right]
```

并将文字改为：较大的 `\varepsilon` 有利于改善打靶问题条件数，逐步减小 `\varepsilon` 后逼近 bang-off-bang 控制。

### 6.5 双模互斥选择

保留并核对了基于 `H_1` 和 `H_2` 的模式选择规则：

```latex
\delta_1^{*}=\begin{cases}
\delta_1, & H_1\leq H_2,\\
0, & H_1>H_2,
\end{cases}
```

```latex
\delta_2^{*}=\begin{cases}
\delta_2, & H_2<H_1,\\
0, & H_2\geq H_1,
\end{cases}
```

并将文字说明改为“保留 Hamiltonian 较小的可行模式”，避免 “truly activated” 这类不够正式的表达。

### 6.6 质量协态方程

核对后保留：

```latex
\dot{\lambda}_m
=-\frac{T_1\delta_1^{*}+T_2\delta_2^{*}}{m^2}\left\|\bm{M}^{\mathrm{T}}\bm{\lambda}_x\right\|
```

该式与 Hamiltonian 中推力加速度项对 `m` 的偏导一致。

## 7. 两点边值问题

位置：主 `.tex` 文件约第 225 到 240 行。

主要修改：

- 将段落改写为更标准的 two-point boundary-value problem 表述。
- 将打靶变量统一写为：

```latex
\bm{z}=\begin{bmatrix}\bm{\lambda}_x^{\mathrm{T}}(t_0)&\lambda_m(t_0)\end{bmatrix}^{\mathrm{T}}
```

- 将打靶函数写为：

```latex
\bm{\Phi}_{F,\mathrm{rend}}(\bm{z})=
\begin{bmatrix}
\bm{x}(t_{\mathrm{f}};\bm{z})-\bm{x}_{\mathrm{f}}(t_{\mathrm{f}})\\
\lambda_m(t_{\mathrm{f}};\bm{z})
\end{bmatrix}=\bm{0}
```

修改目的：避免原文中将 `\bm{\lambda}_x(t_0);\lambda_m(t_0)` 直接写在函数自变量位置导致符号略显繁琐，改用 `\bm{z}` 后更符合打靶法常见写法。

不可关机模式竞争律保持为：

```latex
\delta_1^{*}=\frac{1}{2}\left[1+\tanh\left(\frac{S_1-S_2}{\varepsilon_s}\right)\right],\qquad \delta_2^{*}=1-\delta_1^{*}
```

在该定义下：

- `S_1-S_2>0` 时，模式 1 更有利。
- `S_1-S_2<0` 时，模式 2 更有利。
- `S_1=S_2` 附近由平滑层过渡。

这与公式本身一致。
